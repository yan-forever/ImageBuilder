# ImageBuilder
Used for building images in Minecraft.

用于在Minecraft中建造图片。

# 使用方法：
## 内部准备：
需要在**设置**-**通用**中打开**Websocket**

需要关闭**需要加密的Websocket**

需要作弊和管理员权限
## 外部准备
打开release中的**.jar**文件
输入端口
## 连接
在游戏中输入:
```
/wsserver 127.0.0.1:2333
```
或者
```
/connect 127.0.0.1:2333
```
**注：此处ip和端口仅供参考**

## 使用

按照命令提示使用即可

延迟经测试1ms可以在性能良好的机器上使用，0ms卡死不受支持
